# 💸 Gestor de Gastos

Aplicación web desarrollada con React y Vite que permite registrar gastos y visualizarlos mediante graficas de colores para facilitar el control financiero.

---

## 📌 Características

* Registro de gastos
* Visualización por colores
* Interfaz simple e intuitiva

---

## 🛠️ Tecnologías

* React
* Vite
* JavaScript
* CSS

---

## ⚙️ Cómo ejecutar

```bash
git clone https://github.com/Racoth505/MyA_ProyectoDevOps
cd MyA_ProyectoDevOps
npm install (dentro de la carpeta de Bakcend)
npm install (dentro de la carpeta de Frontend)
npm run dev
```
